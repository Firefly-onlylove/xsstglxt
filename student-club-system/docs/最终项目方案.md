## 二、项目技术架构

```
浏览器 ──HTTP──> scms.exe (Mongoose)
                    │
                    ├── /api/*  → C handlers → MySQL
                    │
                    └── 其他    → 静态文件服务 → web/ (Vue 构建产物)
```

| 层 | 技术 | 说明 |
|----|------|------|
| 前端 | Vue 3 + Vite + Element Plus + Vue Router (Hash) | 构建产物输出到 web/ |
| 后端 | C + Mongoose 7.14 | 单一 scms.exe，嵌入 HTTP 服务 |
| 数据库 | MySQL 8.0 | MySQL C API (libmysql) |
| API 格式 | JSON `{"code":0,"msg":"ok","data":...}` | — |

## 四、C 业务模块开发参考

### 你的每个 .c 文件的标准输入

```c
// 每个 handler 函数的签名统一为：
void xxx_handler(ApiContext *ctx);

// ApiContext 包含（来自 api.h）：
//   ctx->conn      — Mongoose 连接对象
//   ctx->hm        — HTTP 请求消息（含 method/uri/body/query）
//   ctx->user      — 当前登录用户（User 结构体指针，未登录为 NULL）
//   ctx->method    — HTTP 方法字符串（"GET" / "POST" / "PUT" / "DELETE"）
```

### 你能调用的 API 函数（来自 api.h）

| 函数 | 用途 |
|------|------|
| `api_get_param(ctx, "key", buf, size)` | 读取 URL 查询参数或表单字段 |
| `api_get_path_int(ctx, n)` | 读取 URL 路径第 n 段（0-based）作为 int |
| `api_send_json(ctx, 200, json_str)` | 发送 JSON 响应 |
| `api_send_ok(ctx)` | 发送 `{"code":0,"msg":"ok"}` |
| `api_send_data(ctx, json_str)` | 发送 `{"code":0,"data":{json_str}}` |
| `api_send_result(ctx, res)` | 把 MYSQL_RES 转 JSON 数组返回 |
| `api_error(ctx, code, msg)` | 发送错误响应 |
| `api_require_school_admin(ctx)` | 校验当前用户是学校管理员，否则自动返回错误 |
| `api_require_college_admin(ctx)` | 校验学院管理员 |
| `api_require_club_admin(ctx, club_id)` | 校验社团管理员 |

### 你能调用的数据库函数（来自 db.h）

| 函数 | 用途 |
|------|------|
| `MYSQL_RES *db_query("SELECT ... %d ...", id)` | 执行查询，返回结果集（用完需释放） |
| `int db_execute("UPDATE ... %s ...", str)` | 执行增删改，返回影响行数 |
| `int db_query_int("SELECT COUNT(*) ...")` | 返回单个 int 结果 |
| `char *db_error()` | 返回最后一次错误信息 |

### 开发模式模板

```c
#include "../../include/api.h"
#include "../../include/db.h"
#include "../../include/common.h"

// handler 函数 — 由 routes.c 中的路由表调用
void col_dashboard(ApiContext *ctx) {
    // 1. 权限校验
    if (api_require_college_admin(ctx)) return;
    
    // 2. 执行查询
    MYSQL_RES *res = db_query(
        "SELECT COUNT(*) FROM clubs WHERE college_id=%d AND status='approved'",
        ctx->user->college_id);
    
    // 3. 处理结果
    int club_count = db_query_int(
        "SELECT COUNT(*) FROM clubs WHERE college_id=%d AND status='approved'",
        ctx->user->college_id);
    
    // 4. 返回 JSON
    mg_http_reply(ctx->conn, 200, "Content-Type: application/json; charset=utf-8\r\n",
        "{\"code\":0,\"data\":{\"club_count\":%d}}", club_count);
}
```
